Lazy verification

Overview

Lazy verifiers only verify outstanding messages when idle, or when forced(). Lazy verifiers implement the VerifyLazily interface. There are two implementations, VerifyMerkleLazily and VerifyHisttreeLazily that lazily verify Merkle and spliced signatures respectively.

Messages must be submitted before being forced. forceUser(Object) forces all objects who report the user given.

Just as before, messages to be verified must satisfy interface IMessage.
